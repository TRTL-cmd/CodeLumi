# Lumi Beta Testers

Thanks for helping test Lumi Desktop. This beta focuses on stability, onboarding, and core features.

## Goals
- Catch crashes, errors, and regressions
- Validate onboarding and Ollama setup
- Confirm core features work reliably
- Collect focused feedback and suggestions

## System Requirements
- Windows 10/11
- Ollama running locally at http://localhost:11434
  - Install: https://ollama.ai
  - Recommended model: `ollama pull gemma3:4b`

## Install
1) Download the Windows installer: `Lumi Setup 1.0.0.exe`
2) Run the installer and launch Lumi
3) Complete the onboarding steps

## Test Areas
- Onboarding flow (Ollama detection, retry loop)
- Chat responses and stability
- Code sandbox editor
- Self-learning start/pause
- Security curator (review queue loads)

## Feedback
- Report issues with clear steps to reproduce
- Include screenshots if possible
- Note your OS version and whether Ollama is running

## Privacy Notes
- Avoid sharing personal data, private paths, or secrets
- Do not paste real credentials into the app
